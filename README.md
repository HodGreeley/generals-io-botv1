# generals-io-botv1


This is a General.io bot built on https://github.com/greenjoe/sergeants framework.

To run bot:
1.  clone repository
2.  execute _./gradlew build_
3.  execute _./gradlew run_

Before running set your own ID and Username in App.

Enjoy.
